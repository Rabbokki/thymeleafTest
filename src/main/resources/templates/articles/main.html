url method 출력 내용 html page
/articles/list GET 전체 게시글 읽기 성공 list_all.html
/articles/{id} GET 7번 게시글 읽어오기 성공 list_one.html
/articles/create?name=tom&weight=80&height=180 GET 톰의 키는 180, 몸무게는 80입니다. new.html
/articles/update POST 업데이트 성공화면 출력 update_ok.html
/articles/{id}/update GET 7번 게시글 수정 폼 로딩 완료 update.html
/articles/{id}/delete GET 3번 글 삭제 완료 delete_ok.html
/articles/{id}/articleComment POST 11번 게시글의 모든 댓글 보기 성공 comment_view.html
/articles/{id}/articleComments/{article-comment-id}/delete POST 15번 게시글의 3번째 답글 삭제 완료 delete_ok.html
